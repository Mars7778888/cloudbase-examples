## 项目介绍

WebSocket 通常被应用于需要实时通信的场景，包括聊天、实时对战、弹幕等。

本次「小程序 x 云开发」公开课，将通过实战“视频弹幕小程序”开发，教你如何使用「云开发-实时数据推送」能力，快速构建高并发、高可用、弹性扩缩的 WebSocket 应用。
